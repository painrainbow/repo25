# Практическая 3 (Кан, ЭФБО-10-23)
## 1. Динамическая тема
Кнопка в AppBar для переключения светлой/тёмной темы

## 2. Анимации
Увеличение счётчика с анимацией ScaleTransition

Плавное изменение цвета контейнера

Анимация размера иконок

## 3. Взаимодействие
Нажатие на иконки меняет размер и цвет

Поле ввода с мгновенным отображением текста

Карточка для отображения введённого текста

## 4. Улучшенный UI
Card для счётчика с тенью

RoundedRectangleBorder для кнопок

Wrap для адаптивного расположения кнопок

SingleChildScrollView для прокрутки

## 5. Интерактивность
Каждая иконка реагирует на нажатие

Контейнер меняет цвет при нажатии на иконку палитры

Плавающая кнопка для быстрого увеличения счётчика
