# MyShelf (Flutter + Firebase)

Готовый исходный код приложения **MyShelf**:
- Регистрация/вход (Email/Password, Firebase Auth)
- База данных (Firestore): профиль пользователя, полки, предметы коллекции
- Обложки (Firebase Storage)
- Списки: поиск, фильтры, сортировки; списки по полкам

> В архиве **нет** Firebase-конфигов (`google-services.json`, `GoogleService-Info.plist`) — их нужно сгенерировать через `flutterfire configure`.
